# Smart CGPA Calculator

Open `index.html` in any modern browser.

Features:
- Subject GPA calculator
- Weighted semester SGPA calculator
- Weighted overall CGPA calculator
- Fourth Courses tab containing the complete BS Cyber Security curriculum
- Search courses by name or code
- Filter courses by curriculum category
- Responsive design for desktop and mobile

No server, database, PDF, or external library is required.
